ESourceAddressBook — ESource extension for an address book

Description

The ESourceAddressBook extension identifies the ESource as an address book.

Access the extension as follows:

E_SOURCE_EXTENSION_ADDRESS_BOOK

#define E_SOURCE_EXTENSION_ADDRESS_BOOK "Address Book"

Pass this extension name to e_source_get_extension() to access ESourceAddressBook. This is also used as a group name in key files.
